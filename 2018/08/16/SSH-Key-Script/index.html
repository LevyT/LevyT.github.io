Custom Script to add SSH Key to multiple machines

As Devops, or any Server side job with large amount of servers,

Several times we face a point where we want to add a new team member or CI/CD Machine to access specific groups of hosts, this can be frustrating doing it manually and to be honest there is no need to.

That’s why I wrote this bash script

Username is the user that we would like to add the ssh key to, can also be root if root access is allowed through ssh,

Hosts represents all the ip addresses to add the key to.

And “ssh-rsa AAAABBBB Email@example.com” is the actual key

Hope someone will find this helpful.
