Node-Mail-Api

Posted on Sep 9 2018

I Would like to introduce a web application based on NodeJS that I wrote,

Both your browser JavaScript and Node.js run on the V8 JavaScript runtime engine. This engine takes your JavaScript code and converts it into a faster machine code. Machine code is low level code which the computer can run without needing to first interpret it.

In This application The Node Server is running in the back, Express module is serving an index.html file
Which then will be accessed by my browser for that matter,
The Server is set to send according to the given SMTP credentials in Server.js file,

I used here promise for loop in order to achieve the ability to loop asynchronous requests (To Flood)

And after the mail will be sent I will get the response from the server.

Gave it a rich UI (Particles Yay) and Authentication Barrier for the Panel (Admin:Admin - Default Password)

I Perosnally used it on my own state-less SMTP Server to test email spamming capabilities, but it works on gmail and any regular smtp server.
